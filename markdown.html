Indicador de Efetividade

Relatrio de impacto do fomento do CNPq no nmero de publicaes de artigos segundo o Web of Science.

Metodologia

Os dados para a confeco dos grficos foram coletados a partir da plataforma Web of Science, utilizando um filtro especifico para Agencias Financiadoras dentro do ambiente Pesquisas avanas disponvel no prprio site. A extrao de dados foi feita durante a primeira semana de Fevereiro de 2019, e foram considerados apenas os dados de 2008 at 2019, que foram extrados diretamente do site a partir do download de arquivos CSV contendo de duas a trs colunas de dados. Algumas modificaes matemticas, tais como somas de informaes duplicadas e arrendamentos foram feitos para facilitar a analise, alm disso, modificaes estruturais nos arquivos CSV, para corrigir m formataes e facilitar a leitura, tambm foram feitas.
